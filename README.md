# music demo ui

#### 介绍
音乐APP页面

#### 软件架构
一个基于vue3，typescript的移动端音乐视频mv播放器


#### 安装教程

1.  git clone https://gitee.com/rrafa/music-demo-ui.git
2.  npm install

#### 使用说明

1.  npm run serve
2.  npm run build

#### 参与贡献

1.  Fork 本仓库
2.  新建 Feat_xxx 分支



#### 描述
一个基于vue3，typescript的移动端音乐视频mv播放器，仅适用于移动端,后端使用的是基于网易云音乐API搭建的node服务器，服务器接口：http://rrafa.icu:3000

### 后台接口地址
http://rrafa.icu:3000
